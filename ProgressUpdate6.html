⭐ โมดูลใหม่ที่สร้าง 2 Modules

1. Ship Delivery Order — TShip + TShipProd + TShipDet

  • Prisma model: TShip, TShipProd, TShipDet (เพิ่ม TShipDet ในรอบนี้ ตาม Ship_Schema.xlsx)
  • Backend: backend/src/ship-delivery/ — module + service + controller + 3 DTOs
  • API: POST/GET/PATCH/DELETE /api/ship-deliveries + confirm + cancel + Detail CRUD + Aggregate
  • Frontend: Tab UI (รายละเอียดการจัดส่ง / รายการสินค้า) + ปุ่มสร้างสินค้าอัตโนมัติจาก Issue
  • Auto-pull products: ดึง t_isu_prod จากทุก issue ที่อ้างอิง → group productId+uomId → sum qty
  • Wizard 3 steps: Issue Note → เลือก Issue (CONFIRMED) → เลือกลูกค้า

2. A/R Invoice — TArInv (Auto-Create)

  • Prisma model: TArInv + Unique constraint (arInvRefTypIdx + arInvRefId)
  • Backend: backend/src/ar-inv/ — module + service + controller + 3 DTOs
  • Trigger: เมื่อ Sale Invoice ถูก confirm (POST /api/sales-invoices/:id/post)
  • Auto-Create: สร้าง TArInv อัตโนมัติด้วย docStatus = 'CONFIRMED' ใน transaction เดียว
  • Manual create disabled — Frontend View-only + Cancel เท่านั้น
